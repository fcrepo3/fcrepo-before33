Release 2.0 - January 31, 2005

This list covers some of the known problems with Fedora 2.0. Please read this before reporting any new bugs, and watch it regularly (we'll update it as new bugs are found in the release).

All Platforms

  • When running listDatastreams via API-A (SOAP), the list is incomplete, but when running against API-A-LITE (REST), it works fine. The incomplete list consists only of datastreams that are used by disseminators. ( http://www.fedora.info/bugzilla/show_bug.cgi?id=99)

Windows

  • Fedora wsdl incompatible with wsdl.exe ( http://www.fedora.info/bugzilla/show_bug.cgi?id=95)
  • fedora-stop.bat, fedora-ingest-demos.bat may not work if the environment variables JAVA_HOME or FEDORA_JAVA_HOME contain spaces. ( http://www.fedora.info/bugzilla/show_bug.cgi?id=97 and http://www.fedora.info/bugzilla/show_bug.cgi?id=98).
  • fedora-debug.bat contains incorrect path references ( http://www.fedora.info/bugzilla/show_bug.cgi?id=100).
